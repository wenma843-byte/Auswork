# Auswork

A Next.js site for Auswork — professional window cleaning across Brisbane.

## Pages

- **/** – Home
- **/services** – All services
- **/areas** – Service areas
- **/memberships** – Membership plans
- **/reviews** – Customer reviews
- **/contact** – Get a quote

## Run locally

```bash
npm install   # if needed
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Node version

Next.js 16 recommends **Node.js >= 20.9.0**. If you see engine warnings, upgrade Node (e.g. with [nvm](https://github.com/nvm-sh/nvm): `nvm install 20`).
